주제
- okky를 모방한 dokky
- w3school, geeks for geeks, quora, stackoverflow를 참고
- w3school의 코드 실행 페이지를 만들어보자(선택사항)
- 웹 컴파일러를 구현해보자(선택사항)

목표
1. 모든 것을 짜내서 완성해보자.
2. 스프링을 주로 하되 다양한 기술들을 접목해보자.
3. Linux 사용법을 익히자.
4. Linux환경에서 Apache Tomcat을 사용하여 배포를해보자.
5. AWS로 배포를 해보자.

기간
- 2019_02_10~2019_02_28 (개발기간 3주) 주말포함

인원

- 로컬상 Eclipse로 개발을 하며 진행
- VMwere, CentOS를 이용한 Linux에서 Apache Tomcat  배포

- 일단 View단은 JSP진행함

 - okky 정보 부분 참고
